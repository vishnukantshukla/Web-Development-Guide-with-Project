HTML Elements:-
- Everything from the starting tag till ending tag.
- For example:-
<starttag>......</starttag>
- <html>...</html>
- <title>Document</title>
- <body>...</body>

Types of Elements:-
1. Root Element:-
- That tells the system starting of html document.
- Example:-
<html>...</html>
2. Nested Element:-
- Element inside another element
- <html>
<head></head>
</html>
-<head>
<title>..</title>
</head>

3. Empty Element
- Single Tag
- No content
For Example:-
- <hr>
- <br>
